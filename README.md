# 📊 AI-Generated Sales Dashboard

> Interactive sales dashboard built using Claude AI — designed and deployed without manual front-end coding.

---

## 🎯 Project Overview

This project demonstrates how AI can accelerate data product development. 
A fully interactive sales dashboard was designed and deployed using **Claude AI (Cowork)** — from raw data to a working visual dashboard —using plugin data- without writing front-end code manually.

---

## 🚀 Workflow

Raw Data File (CSV/Excel)
        ↓
Uploaded to Claude Cowork 
        ↓
Installed Plugin Data
        ↓
Configured dashboard requirements (KPIs, filters, chart types) 
        ↓
Claude generated interactive HTML dashboard
        ↓
Reviewed, refined, and deployed
---

## Key Takeover
Traditional dashboard development requires front-end skills and hours of coding. 
This project shows that with the right AI workflow, a data analyst can go from raw data to a polished, interactive dashboard in a fraction of the time.
